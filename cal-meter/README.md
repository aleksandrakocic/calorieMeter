# cal-meter

 Created a Vue-based calorie meter
 The meter look  like in the picture (https://ibb.co/fNAZvo)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files

```
npm run lint
```
